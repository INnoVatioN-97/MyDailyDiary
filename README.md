    -   은정이가 상자에서 꺼낸 홍대의 휴대폰을 열어봤을 때 그때 그 장면
    -   ![img_1.png](img_1.png)
## 기능
-   하루 일과를 마무리하는 일기장을 작성할 수 있다.
    -   일기에는 오늘 하루를 대표하는 사진 하나를 넣어야 한다.
-   일기들은 모두 firebase 에 저장되어 로그인시 데이터가 유지되며, 앱 잠금으로 보호된다.
